Right Thrust Gauge Test

-100 (Reverse) 0 +100 (Forward)

How This Works:

  • The slider updates localStorage.setItem('game_rightThrust', value)
  • The SVG loads thrust-right.js internally
  • The thrust-right.js script polls localStorage every 50ms
  • When it detects a change, it moves the indicator line horizontally
  • Range: -100 (full reverse) to +100 (full forward)
