Compass Gauge Test

0° 0° 360°

How To Update

  • Make your instruments illustration in Figma/Illustrator (vector only) and save as .SVG
  • In Illustrator, Save As SVG In SVG Options Screen, Click Button "SVG Coder..."
  • This opens the code in Text Edit. Select All, & Copy
  • Open this html file in TextEdit. Paste your SVG to replace the existing SVG in the indicated area.
  • Remember: In TextEdit > Settings > Open and Save. Uncheck "Display HTML files as HTML code instead of formatted text"

How This Works:

  • There is a compass.svg file. For the purposes of this demo, the student pastes the SVG contents in this HTML page
  • The HTML loads compass.js which contains the update_compass() function
  • The slider updates localStorage.setItem('game_compass', value)
  • After page load, setInterval calls update_compass() every 50ms
  • When it detects a change, it rotates the compass in the SVG
  • Range: 0° = North, 90° = East, 180° = South, 270° = West
