Compass Gauge Test

0° 0° 360°

How This Works:

  • The slider updates localStorage.setItem('game_compass', value)
  • The SVG loads compass.js internally
  • The compass.js script polls localStorage every 50ms
  • When it detects a change, it rotates the compass in the SVG
  • 0° = North, 90° = East, 180° = South, 270° = West
