Rudder Gauge Test

-100 (Left) 0 +100 (Right)

How This Works:

  • The slider updates localStorage.setItem('game_rudder', value)
  • The SVG loads rudder.js internally
  • The rudder.js script polls localStorage every 50ms
  • When it detects a change, it moves the indicator line horizontally
  • Range: -100 (full left) to +100 (full right)
